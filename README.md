# Дамка — шашки нового уровня

> Веб-платформа для игры в русские шашки с AI-тренером, сюжетным режимом, мультиплеером по ссылке и системой лутбоксов.

**Один файл. Без бэкенда. Открыл в браузере — играешь.**

🎮 **Live**: https://prismatic-faun-0d3d28.netlify.app/

---

## Что это и для кого

Большинство сайтов для игры в шашки выглядят так, будто их сделали в 2008 году: серый интерфейс, баннеры, никакой мотивации возвращаться. **Дамка** переосмысляет жанр — это полноценный игровой сервис уровня мобильных хитов, который при этом умещается в один HTML-файл.

**Для кого:**
- Игроки, которым надоели монотонные «онлайн-шашки» — здесь есть прогрессия, награды и причина возвращаться каждый день.
- Дети и новички — AI-тренер на 5 уровнях сложности и говорящие персонажи-помощники объясняют ходы в дружелюбной форме.
- Любители казуальных дуэлей — режим Блиц 3 минуты и игра с друзьями по ссылке без регистрации.
- Геймеры, ценящие «жопу-в-кресло»-механики — лутбоксы с рулеткой, скины разной редкости, инвентарь и сюжетная карта мира из 4 локаций.

## Главные фичи

### 🤖 ИИ на 5 уровнях
Минимакс с альфа-бета отсечением и оценочной функцией (материал, продвижение, центр, задний ряд). Глубина 1→6, рейтинг 600→1800. Время хода ограничено 1.5 сек.

### 📊 AI Coach
Пост-анализ каждой партии: каждый ход размечается лейблом (Лучший / Хороший / Неточность / Ошибка / Зевок), считается % точности, выводится топ моментов для разбора с объяснениями вроде *«Сильнее было c5:e7 — было взятие 2 шашек»*.

### 🔗 Мультиплеер по ссылке
WebRTC через [PeerJS](https://peerjs.com) — без своего бэкенда. Хост создаёт комнату, гость заходит по `?room=ID`. Никнеймы и аватары синхронизируются.

### ⚔️ Сюжетный режим
4 уникальных мира: 🌲 Лесная деревня → 🏜️ Пески пустыни → 🏰 Ледяной замок → 🌌 Космическая цитадель. В каждом — 5 уровней-противников и финальный босс. Карта с SVG-путями, узлами и анимациями. Рейтинг противников растёт с 600 до 2500. Победа над боссом открывает следующий мир и даёт **золотой кейс**.

### 🎁 Лутбоксы с рулеткой (CS:GO-style)
- 3 типа коробок: 📦 картонная, 🎁 серебряная, 👑 золотая
- Шансы открыто показаны цветной полосой с процентами
- При открытии — горизонтальная прокрутка из 48 скинов с тикающим звуком и плавным замедлением на cubic-bezier
- Выигрышный скин подсвечивается своим цветом редкости и мигает

### 🎒 Инвентарь со скинами
22 уникальных скина: **12 для шашек** (Классика, Камень, Бронза, Серебро, Океан, Лес, Мрамор, Кристалл, Галактика, Неон, Магма, Золото) и **10 для досок** (Дерево, Картон, Сосна, Мята, Сапфир, Тёмный мрамор, Королевский, Вулкан, Кибер, Алмаз). Каждый — со своей CSS-визуализацией. Подсветка по редкости:

- ⚪ **Common** — серая
- 🟢 **Rare** — зелёная
- 🟣 **Epic** — фиолетовая
- 🔴 **Mythic** — красная
- 🟡 **Legendary** — золотая (пульсирует)

### 🎭 8 говорящих персонажей-тренеров
Случайный персонаж сопровождает каждую партию и даёт подсказки в своём стиле: 🦊 Хитрый Лис, 🦉 Сова-Профессор, 🐸 Кваклогик, 🐢 Мудрая Черепаха, 🐲 Огнедышащий, 🦄 Радужный Единорог, 😼 Босс-Кот, 🧙 Маг Шашкин. Например, на одну и ту же ситуацию (доступно взятие 3 шашек):
- 🐲 Дракон: «3 ВРАГОВ В ОГОНЬ!! 🐉🔥🔥»
- 😼 Кот: «комбо на 3? даже не вспотел.»
- 🦄 Единорог: «Какая красивая комбинация на 3 шашек ✨»

### ⚡ Блиц 3 минуты
Чесс-клок на каждого игрока. Закончилось время — проиграл. Пульсация таймера при <30 сек.

### 🏆 Социальный слой
Городской и глобальный лидерборд с 30+ предзаполненными игроками из СНГ. ELO-стиль рейтинг с +18 / −12 за победу/поражение.

### ✨ Дамка Pro
Mock Stripe-чекаут, премиум-фичи: расширенный AI Coach, бонусные коробки за серию побед, облачное сохранение (планируется).

### 🌗 UX-мелочи
Тёмная/светлая тема, синтезированные звуки через Web Audio API, подсказки обязательных шашек, история ходов в нотации (a3-b4, c5:e7:g5), отмена хода, превью последнего хода, мобильная адаптация (от 360px).

### 🎯 Ежедневные пазлы
8 хардкорных позиций, пазл дня выбирается по дате. Streak-счётчик с рекордом. За решение — серебряный кейс. Безопасно от спама — учитывается только один пазл в сутки.

### 💬 Чат + 👥 Друзья + 🏆 Турниры
В онлайн-партии — текстовый чат через PeerJS data channel плюс быстрые сообщения. После игры можно добавить соперника в друзья по PeerID и приглашать одним кликом. Локальные турниры на 4 игроков — полуфиналы и финал, все матчи играются за устройством.

### 🌐 i18n (RU/EN/KK)
Переключатель языка в профиле — русский, английский, қазақша. Покрыты ключевые строки UI.

### 📱 PWA-приложение
manifest.json + service worker + SVG-иконка. Можно установить на телефон с экрана «Установить как приложение». Работает офлайн (партии против ИИ, инвентарь, пазлы). Мультиплеер требует интернета.

### ☁️ Облачный лидерборд
Опциональная интеграция с Firebase Realtime DB через REST — без зависимостей. Включается в профиле: пользователь вставляет URL своего Firebase-проекта, и результаты после побед уходят в облако. Без конфига работает локальная версия с засеянными игроками.

## Стек

| Слой | Технология | Зачем |
|---|---|---|
| UI | HTML5 + CSS3 (custom properties, grid, animations) | Лёгкий, быстрый, без сборщика |
| Логика | Vanilla JS (ES2022) | ~3600 строк в одном файле, нулевые зависимости |
| ИИ | Минимакс + α-β + транспозиционная таблица (Zobrist) + iterative deepening | <10 мс/ход на глубине 6, 8/8 побед над random |
| P2P | [PeerJS 1.5](https://peerjs.com) | WebRTC без своего сигнального сервера |
| Звук | Web Audio API (синтез без файлов) | Один HTML без ассетов |
| Хранилище | `localStorage` | Профиль, инвентарь, прогресс, лидерборд |
| Графика | CSS-градиенты, SVG | Каждый скин — это набор `linear-gradient` |

**Никаких npm, build, бандлеров, бэкендов.** Файл `index.html` весит 157 КБ и работает офлайн (кроме мультиплеера, которому нужен интернет для STUN/сигналинга).

## Архитектура кода

Один HTML-файл организован в логические секции:

```
index.html
├── <style> ~600 строк CSS (темы, доска, фигуры, модалки, рулетка, карта)
└── <script>
    ├── Constants & helpers (RARITY, BOX_TYPES, SKIN_CATALOG, COACHES, WORLDS)
    ├── State & Storage (профиль, миграции, localStorage)
    ├── Game logic (правила, генерация ходов, мульти-взятия)
    ├── AI (минимакс + альфа-бета + оценка)
    ├── AI Coach (пост-анализ, разметка ходов)
    ├── Sound (Web Audio синтез)
    ├── UI rendering (меню, игра, инвентарь, кампания, лидерборд)
    ├── Multiplayer (PeerJS host/guest)
    ├── Timers (блиц)
    ├── Box opening (рулетка-анимация)
    └── App router (state.view + методы App.*)
```

Игровая логика отделена от UI и покрыта **14 unit-тестами** (правила взятий, мульти-прыжки, превращение в дамку посреди серии, обязательность взятий, дамка-летучка).
